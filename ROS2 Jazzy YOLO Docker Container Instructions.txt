Steps to make the docker container work:

git clone https://github.com/mac785/ROS2-Dice-Roller

run: docker build -t yolo_ros .

run: docker compose exec ros bash

You are now operating inside the docker container

to echo the topic:
	ros2 topic echo /dice/sum

to test the joysticks:
	jstest /dev/input/js0

to use buildkit and cached downloads:
	DOCKER_BUILDKIT=1 docker build -t my_ros2_image .
